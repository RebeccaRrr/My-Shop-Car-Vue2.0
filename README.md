# My-Shop-Car-Vue2.0

《使用vue2.0实现购物车和地址选配功能》

Imooc课程中的配课练习题

由灿灿根据题目另辟思路制作，并添加注释，现开源供大家学习。

# 使用注意事项

购物车列表为跨域请求文件，并且对文件的跨域请求部分浏览器并不支持，故访问页面请用火狐浏览器打开
